The outer looks a.k.a.
frontend.

Other than anything what catches peoples eyes are the visuals of a site. I provide that by HTML,CSS,JS. But in this fast paced world we need things to be ready faster. For that I have learnt libraries like JQuery,AngularJS,BootStrap,SASS. Most powerful library that I know right now is ReactJS which will create your application in much lesser space/memory(LOC).With that I also know Redux which is part of React's state management system. These will help to create a simple,understandable but still a sophisticated platform for your business.

The inner functioning a.k.a. backend.

Not only frontend,But A site requires a backend as well and I provide it by using technologies like NodeJS/ExpressJs which works on single thread,in simple words do not causes huge load on your system. For your Database, I use the best No-SQL system there i.e. MongoDB/Atlas .Don't worry if things need to be rendered from the backend only, for that I know EJS and for smaller projects I have Firebase as database for you. In all this, I make use of great NPM libraries like bcrypt,JWT,multer,passport,nodemailer ,nodecron and many more. This helps your site provide utmost security and a managed system.
